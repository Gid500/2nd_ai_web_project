## Frontend 디렉토리 용도

이 프로젝트의 `frontend` 디렉토리는 React 기반의 웹 애플리케이션을 포함합니다. 주요 디렉토리의 용도는 다음과 같습니다.

### `frontend/src` 디렉토리

-   **`asset/`**: 이미지, CSS 등 정적 자원들을 관리하는 디렉토리입니다.
    -   **`asset/css/`**: 애플리케이션의 전반적인 스타일과 각 컴포넌트 및 페이지별 스타일을 정의하는 CSS 파일들을 포함합니다.
-   **`common/`**: 애플리케이션 전반에 걸쳐 재사용되는 공통 컴포넌트 및 유틸리티 함수들을 모아둔 디렉토리입니다.
    -   **`common/hook/`**: 사용자 입력 처리와 같은 특정 로직을 캡슐화한 커스텀 React 훅들을 포함합니다.
-   **`pages/`**: 애플리케이션의 각 웹 페이지를 구성하는 컴포넌트들을 담고 있는 디렉토리입니다. 각 하위 디렉토리는 특정 페이지(예: `home`, `posts`, `signin`)와 관련된 모든 파일(메인 컴포넌트, API 호출 로직, 하위 컴포넌트, 커스텀 훅 등)을 포함합니다.
    -   **`pages/about/`**: 'About Us' 페이지와 관련된 컴포넌트 및 로직을 포함합니다.
    -   **`pages/admin/`**: 관리자 페이지와 관련된 컴포넌트 및 로직을 포함합니다.
    -   **`pages/error/`**: 403 Forbidden, 404 Not Found와 같은 에러 페이지 컴포넌트들을 포함합니다.
    -   **`pages/home/`**: 메인 페이지와 관련된 컴포넌트 및 로직을 포함합니다.
    -   **`pages/posts/`**: 게시글 목록, 상세 보기, 작성 등 게시글 기능과 관련된 컴포넌트 및 로직을 포함합니다.
        -   **`pages/posts/components/`**: 게시글 페이지 내에서 사용되는 하위 컴포넌트들을 포함합니다.
    -   **`pages/signin/`**: 로그인 페이지와 관련된 컴포넌트 및 로직을 포함합니다.
    -   **`pages/signup/`**: 회원가입 페이지와 관련된 컴포넌트 및 로직을 포함합니다.
-   **그 외 주요 파일**: 
    -   **`App.jsx`**: 애플리케이션의 메인 진입점 역할을 하며, 라우팅 설정 및 전역 레이아웃을 정의합니다.
    -   **`index.jsx`**: React 애플리케이션을 DOM에 마운트하는 초기 설정 파일입니다.
    -   **`App.css` / `index.css`**: 각각 `App.jsx` 컴포넌트와 전역 스타일에 적용되는 CSS 파일입니다.
